# Performance Testing Reference

## Goals
- Measure latency, throughput, error rate, and saturation.

## Suggested Levels
- Smoke perf: quick response-time baseline.
- Load test: expected traffic profile.
- Stress test: identify degradation and breaking points.

## Metrics to Report
- p50/p95/p99 latency.
- Requests per second.
- Error percentage.
- Resource usage (CPU, memory, DB connections).
